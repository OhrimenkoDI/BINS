# Расчёт угловых скоростей между двумя кватернионами

## 1. Исходные данные и соглашение

Известны:

- `q0` — начальная ориентация Body FRD → NED;
- `q1` — конечная ориентация;
- `T` — требуемое время перехода;
- `q0` и `q1` нормализованы.

Требуется получить одновременные задания угловой скорости в связанных осях:

`ωb = (p, q, r)`,

где `p` — крен вокруг Body X, `q` — тангаж вокруг Body Y, `r` — рыскание вокруг Body Z.

## 2. Относительный кватернион

Так как угловые скорости заданы в локальных осях аппарата:

`qe = q0⁻¹ ⊗ q1`.

Отсюда:

`q1 = q0 ⊗ qe`.

Для единичного кватерниона `q0 = (w0,x0,y0,z0)`:

`q0⁻¹ = q0* = (w0,−x0,−y0,−z0)`.

Кватернионы `q` и `−q` задают одинаковую ориентацию. Для кратчайшего поворота:

`qe.w < 0  ⇒  qe := −qe`.

## 3. Единая ось и угол поворота

Пусть:

`qe = (w,x,y,z)`.

Длина векторной части:

`v = sqrt(x²+y²+z²)`.

Полный угол кратчайшего поворота:

`θ = 2·atan2(v,w)`.

Единичная ось:

`e = (x/v, y/v, z/v)`.

Если `v` близко к нулю, поворот отсутствует и угловая скорость принимается равной нулю.

## 4. Вектор поворота по осям Body

Вектор поворота:

`φ = θ·e = Log(qe)`.

Компоненты:

`φx = θ·x/v`,

`φy = θ·y/v`,

`φz = θ·z/v`.

В Body FRD:

- `φx` — компонента вокруг X, крен;
- `φy` — компонента вокруг Y, тангаж;
- `φz` — компонента вокруг Z, рыскание.

Это не три последовательных угла Эйлера. Компоненты совместно задают один поворот вокруг единой оси, причём `|φ| = θ`.

## 5. Постоянная угловая скорость

Для перехода за время `T` с постоянной скоростью:

`ωb = φ/T`.

То есть:

`p = φx/T`,

`q = φy/T`,

`r = φz/T`.

Итоговая формула:

`[p,q,r]ᵀ = Log(q0⁻¹⊗q1)/T`.

Результат измеряется в радианах в секунду. Линейная по времени траектория

`q(t) = Slerp(q0,q1,t/T)`

соответствует этому постоянному вектору угловой скорости.

## 6. Плавный старт и остановка

Постоянная скорость создаёт скачки `0 → ω → 0`. Для плавного перехода:

`u = t/T`,

`s(u) = 10u³ − 15u⁴ + 6u⁵`,

`q(t) = Slerp(q0,q1,s(u))`.

Производная параметра:

`ds/dt = (30u² − 60u³ + 30u⁴)/T`.

Угловые скорости:

`ωb(t) = φ·ds/dt`,

`p(t) = φx·ds/dt`,

`q(t) = φy·ds/dt`,

`r(t) = φz·ds/dt`.

Ось вращения остаётся неизменной, плавно меняется только модуль скорости. Максимум при `u=0.5`:

`|ω|max = 1.875·θ/T`.

## 7. Выбор времени по ограничениям

Для постоянной скорости и отдельных ограничений:

`T ≥ max(|φx|/pmax, |φy|/qmax, |φz|/rmax)`.

Для плавной S-кривой:

`T ≥ 1.875·max(|φx|/pmax, |φy|/qmax, |φz|/rmax)`.

При ограничении полного модуля:

- постоянная скорость: `T ≥ θ/ωmax`;
- S-кривая: `T ≥ 1.875·θ/ωmax`.

## 8. Практическое управление

Расчёт постоянного `(p,q,r)` задаёт идеальную траекторию, но реальный аппарат не выполняет её абсолютно точно. Поэтому во время движения желательно пересчитывать остаточную ошибку по текущему кватерниону:

`qe(t) = q(t)⁻¹ ⊗ qref(t)`.

Затем используется закон:

`ωcmd = ωff + KR·Log(qe)`.

`ωff` задаёт движение по расчётной траектории, а обратная связь устраняет фактическую ошибку.

## 9. Краткий алгоритм

```text
qe = inverse(q0) × q1
если qe.w < 0: qe = -qe
v = length(qe.xyz)
θ = 2·atan2(v, qe.w)
φ = θ·qe.xyz/v
ω = φ/T
```

Итог:

`(p,q,r) = Log(q0⁻¹⊗q1)/T`.
