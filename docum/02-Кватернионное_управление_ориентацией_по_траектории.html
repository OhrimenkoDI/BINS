Кватернионное управление ориентацией по траектории

1. Постановка задачи

На уровне H0 аппарат имеет произвольную ориентацию q0; на уровне Hh требуется получить qh. Управление задаётся единым вектором угловой скорости в Body FRD: ωcmd=(p_cmd,q_cmd,r_cmd).

высота/время → qref → ошибка qe → (p,q,r) → контур гироскопов

2. Slerp как генератор траектории

qref(s) = Slerp(q0,qh,s),   s∈[0,1]

Slerp проходит кратчайшую геодезическую и задаёт один результирующий поворот. Если dot(q0,qh)<0, следует заменить qh:=−qh, поскольку q и −q обозначают одинаковую ориентацию.

3. Плавный прогресс

s = clamp((H−H0)/(Hh−H0),0,1)

Чтобы исключить скачки скорости и ускорения на границах:

τ(s)=10s³−15s⁴+6s⁵,   qref=Slerp(q0,qh,τ)

Высоту необходимо фильтровать. При остановках или возврате по высоте предпочтителен монотонный параметр прогресса либо временная траектория с коррекцией по высоте.

4. Кватернион ошибки

Для соглашения q=q_nb, Body FRD → NED:

qe = q−1 ⊗ qref,   qref=q⊗qe

Если qe.w<0, применяется qe:=−qe, чтобы ошибка представляла поворот не более 180°.

5. Единый вектор ошибки вращения

Пусть qe=(w,v). Тогда:

θ=2·atan2(|v|,w),   e=v/|v|,   φ=θe=Log(qe)

Компоненты φ=(φx,φy,φz) выражены в Body FRD. Для малой ошибки возможно приближение φ≈2v, но для произвольной ориентации используется полная формула.

6. Задание угловой скорости

ωcmd = KRφ
p_cmd = Kx · φx
q_cmd = Ky · φy
r_cmd = Kz · φz

Это компоненты одного вектора, а не скорости углов Эйлера. Для точного слежения за движущимся заданием:

ωcmd = ωff + KRφ

7. Ограничение скорости

Ограничивается норма единого вектора: |ωcmd|≤ωmax. При превышении:

ωcmd := ωcmd · ωmax/|ωcmd|

Так сохраняется ось вращения. При разных пределах приводов применяется эллипсоид: (p/pmax)²+(q/qmax)²+(r/rmax)²≤1.

8. Внутренний контур

τcmd = Kω(ωcmd−ωmeas)

Внутренний регулятор использует гироскоп и учитывает динамику, ограничения приводов, интегральную составляющую и anti-windup.

9. Связь с траекторией полёта

Крен и тангаж меняют направление тяги. Навигационный контур должен формировать физически достижимый qref и сохранять достаточную вертикальную составляющую тяги.

навигация → допустимый qref → регулятор SO(3)
           → p/q/r → контур скорости → приводы

10. Итоговый закон

Связанный документ: 03 — Расчёт угловых скоростей между двумя кватернионами описывает переход за фиксированное время и может служить генератором qref(t) и ωff(t) для этого регулятора.

qref = Slerp(q0,qh,τ)
qe = q−1 ⊗ qref
ωcmd = ωff + KRLog(qe)

Slerp задаёт ориентационную траекторию, Log(qe) — единую локальную ось и угол ошибки, а регулятор формирует единый вектор заданной угловой скорости.
