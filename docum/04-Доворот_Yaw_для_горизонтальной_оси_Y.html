Доворот Yaw для горизонтального положения оси Y

1. Постановка задачи

Используются Body FRD и NED; q=q_nb переводит Body → NED. Yaw выполняется вокруг локальной Body Z, а приращение применяется справа: qnew=q⊗DeltaQz.

Yaw не выравнивает платформу: он не меняет ось Z. Он может сделать горизонтальной одну ось, совместив её с линией пересечения плоскости платформы и горизонтальной плоскости.

Обозначение: специальный доворот называется ψ_align. Он не совпадает с Euler Yaw ψ текущей ориентации из документа 05; это отдельное управляющее приращение вокруг локальной Z.

2. Вертикальные компоненты

Xn=q⊗Xb⊗q−1
Yn=q⊗Yb⊗q−1

Обозначим XD=Xn.Z и YD=Yn.Z. В NED горизонтальная ось имеет нулевую компоненту Z.

3. Вывод угла

После локального доворота:

Y'n=−sin(ψalign)Xn+cos(ψalign)Yn
Y'D=−XD·sin(ψalign)+YD·cos(ψalign)

Из условия Y'D=0:

ψalign = atan2(YD, XD)
XD = oX3.z
YD = oY3.z
YawAlign = atan2(YD, XD)
DeltaQz = Quaternion((0,0,1), YawAlign)
qnew = q × DeltaQz

4. Два решения

ψ_align и ψ_align+π оба делают Y горизонтальной, но направляют её противоположно. Для минимального доворота:

ψ_align = atan2(YD, XD)
if ψ_align >  π/2: ψ_align = ψ_align − π
if ψ_align < −π/2: ψ_align = ψ_align + π

Если требуется конкретное направление +Y, решение выбирается по скалярному произведению с заданным горизонтальным направлением.

5. Горизонтальная платформа

При XD=YD=0 ось Y горизонтальна при любом Yaw. Поэтому перед atan2 проверяется sqrt(XD²+YD²)<ε; результатом становится ψ_align=0.

6. Формула через компоненты кватерниона

Для нормализованного q=(w,x,y,z):

XD=2(xz−wy)
YD=2(yz+wx)
ψalign = atan2(yz+wx, xz−wy)

В произведениях выше x, y, z и w являются компонентами кватерниона.

7. Проверка знака

Формула соответствует qnew=q⊗DeltaQz. Если привод считает положительный Yaw противоположно, используется −ψ_align. Для проверки платформу наклоняют вокруг X: при YD≠0 и XD≈0 доворот будет близок к ±90°.

8. Итог

ψalign=atan2(Yn.Z, Xn.Z)

ψalign=atan2(yz+wx, xz−wy)
